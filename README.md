# QuizMaster 🎓

QuizMaster is an interactive programming quiz web app where users can test their knowledge in Java, Python, JavaScript, DSA, and General Knowledge.

## 🚀 Features

- Multiple quiz categories
- Timed quiz questions
- Score tracking
- Best score leaderboard
- Correct and incorrect answer feedback
- Local stats saving
- Clean dark UI
- Responsive design

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- CSS
- LocalStorage
